### Binary Tree

- Basic concepts:
  - Node
  - Sub-Trees
  - Parent node
  - Ancestors of a node
  - Leaf node
- Construct Binary Tree:
  - from PreOrder
- Traversal Binary Tree:
  - PreOrder Traversal
    - Root
    - Left subtree
    - Right subtree
  - InOrder Traversal
    - Left subtree
    - Root
    - Right subtree
  - Post order
    - Left subtree
    - Right subtree
    - Root
  - Level Order Traversal
    - Each level from root along the height
